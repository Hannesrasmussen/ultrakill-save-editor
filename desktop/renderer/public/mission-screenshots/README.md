# Mission Screenshots

Mission card screenshots used by the Missions UI.

## Naming

Use lowercase mission code with `.png`:

- `<mission-code>.png`

Examples:

- `0-1.png`
- `2-s.png`
- `p-1.png`

## Notes

- Keep filenames lowercase.
- The UI can probe additional formats (`webp`, `jpg`, `jpeg`), but this project standardizes on `png`.
- Most images are game-derived assets and are not covered by the repository MIT license.
